# Unemployment rate

**DRAFT**

```
unrate/
    data/
        national/
            2010-2019.csv 
            2000-2009.csv
```

Each dataset records the unemployment rate published by the Bureau of Labor
Statistics in the time period indicated by the name of the file. The data are
formatted as:

```
date,value
2016-06-01,4.9
2016-07-01,4.9
2016-08-01,4.9
2016-09-01,4.9
2016-10-01,4.8
2016-11-01,4.6
2016-12-01,4.7
2017-01-01,4.8
```
